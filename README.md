# operating-system
This repo contains C programs for OS algorithms including CPU Scheduling, Page Replacement, Deadlock Avoidance, and Memory Allocation.

# Operating System Lab Programs

This repository contains C programs for important Operating System algorithms and concepts.

# Programs Included
First Come First Served (FCFS) CPU Scheduling
Non-Preemptive Shortest Job First (SJF) Scheduling
Non-Preemptive Priority Scheduling
Round Robin CPU Scheduling
Banker’s Deadlock Avoidance Algorithm
FIFO Page Replacement Algorithm
LRU Page Replacement Algorithm
First Fit Memory Allocation
Best Fit Memory Allocation
Technologies Used
C Programming
GCC Compiler

# Purpose

These programs are implemented for understanding and practicing core Operating System concepts and lab experiments.
